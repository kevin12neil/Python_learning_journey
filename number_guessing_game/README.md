# Number Guessing Game

A number guessing game built using Python.

## Features
- Easy and hard difficulty modes
- Replay functionality

## Workflow
- The program randomly selects a number between 1 and 100
- Player chooses a difficulty level
- Player keeps guessing until:
  - the correct number is guessed
  - attempts run out

## Limitations
- Does not validate non-numeric input

## How to Run

```bash
python number_guessing.py
```
