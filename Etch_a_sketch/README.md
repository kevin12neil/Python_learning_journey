# Etch A Sketch

A simple Etch A Sketch drawing program built using Python Turtle graphics.

## Controls
- `W` → Move forward
- `S` → Move backward
- `A` → Turn left
- `D` → Turn right
- `C` → Clear screen

## Features
- Interactive keyboard controls
- Screen reset functionality

## Concepts Used
- Event listeners
- Keyboard input handling

## How to Run

```bash
python etch_a_sketch.py
```
