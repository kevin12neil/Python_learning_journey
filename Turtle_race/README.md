# Turtle Race Game

A simple turtle racing game built using Python Turtle graphics.

## Features
- Multiple racing turtles
- Randomized movement

## Workflow
- User places a bet on a turtle color
- Turtles move forward randomly
- First turtle to cross the finish line wins
- Result is displayed to the user

## Limitations
- No user input validation

## How to Run

```bash
python turtle_race.py
```
