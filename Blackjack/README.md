# Blackjack Game

A command-line blackjack game built using python.

## Features
- Hit / Stand gameplay
- Dealer automation
- Ace value adjustment(1/11)
- Blackjack detection
- Bust detection
- Replay functionality

## Workflow
- Player and dealer are dealt 2 cards
- Player chooses to hit or stand
- Dealer automatically draws until reaching 17 or higher
- Winner is decided based on blackjack rules

## Limitations
- Uses simplified card generation instead of a full 52 card deck
- Does not support betting or splitting

## How to Run

```bash
python blackjack.py
```
