# Blind Auction simulation
Simulates a blind auction scenario where each participant places a bid without knowing others' bids.

## Workflow
- Accepts multiple bidders
- Stores bids using a dictionary
- Determines the highest bidder


## Limitations 
- Screen clearing is simulated using newlines
- Does not handle duplicate bidders
- Invalid bids are not checked 

## How to Run

```bash
python blind_auction.py
```
